# Alerting Rule Exports

Kibana rule exports (NDJSON) should be placed in this folder. Example filenames:

- `high-cpu.ndjson`
- `disk-space-critical.ndjson`
- `memory-pressure.ndjson`
- `postgres-connections.ndjson`
- `redis-memory.ndjson`
- `nginx-5xx-spike.ndjson`

Export rules from Kibana after creating them so they can be version-controlled here.
